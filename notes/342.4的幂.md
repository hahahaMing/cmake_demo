# 342. 4的幂
给定一个整数，写一个函数来判断它是否是 4 的幂次方。如果是，返回 `true` ；否则，返回 `false` 。

整数 `n` 是 4 的幂次方需满足：存在整数 `x` 使得 `n == 4 ^ x`

 

**示例 1：**

**输入：**n = 16
**输出：**true


**示例 2：**

**输入：**n = 5
**输出：**false


**示例 3：**

**输入：**n = 1
**输出：**true




**提示：**


- `-2 ^ 31 <= n <= 2 ^ 31 - 1`




**进阶：**


- 你能不使用循环或者递归来完成本题吗？


## 我的代码

```c++
class Solution {
public:
    bool isPowerOfFour(int n) {
        if(n<=0)return false;
        int a = n & 0b01010101010101010101010101010101;
        if(a!=n)return false;
        if((a&(a-1))==0){
            return true;
        }
        return false;
    }
};
```

## 题解

```c++
class Solution {
public:
    bool isPowerOfFour(int n) {
        return n > 0 && (n & (n - 1)) == 0 && (n & 0xaaaaaaaa) == 0;
    }
};
```
